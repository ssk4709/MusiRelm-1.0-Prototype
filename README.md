This ReadME file explains all the web-pages created for the project in short. Consider it the guide of the project.

This website is only a STATIC FRONT-UI REPRESENTATION

This prototype version contains :

Home.html - This is the main webpage where user comes.
Artist.html - This section contains the total artists that are currently on the website.
Genre.html - This section contains the total genres of music that are currently there on the website

<artist name>( AP ).html - It is the artist page where the user gets info + access to albums about a certain artist.
<album name>.html - It is the info page + musci playing webpage where the user gets the info of the particular album and can play the music.
